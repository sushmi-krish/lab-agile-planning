# lab-agile-planning
This repository contains my Agile Lab project, where I practised real Agile project‑management workflows using GitHub Agile tools. The lab demonstrates how to plan, organise, and track work using a Kanban board, user stories, backlogs, and sprint events.
-------------------------------------------------------------------------------------------------------------------------------------
## Project Overview
The goal of this lab is to simulate a real Agile project environment by creating a structured workflow that supports planning, prioritisation, and continuous delivery. The project includes user stories, acceptance criteria, backlog management, and sprint activities.
___________________________________________________________________________________________________________________________________
## Agile Practices Implemented
### Kanban Board Setup
* Created a full Kanban board using GitHub Projects.
* Configured workflow columns (Backlog → In Progress → Review → Done).
* Added labels, priorities, and task assignments.
* Tracked progress and updated task status throughout the workflow.
_____________________________________________________________________________________________________________________________________
# Backlog Management
* Created and organised the Product Backlog with prioritised user stories.
* Managed the Pipeline Backlog to prepare items for upcoming sprints.
* Refined backlog items to ensure they were ready for development.
_____________________________________________________________________________________________________________________________________
# Sprint Events
* Conducted Sprint Planning by selecting backlog items for the sprint.
* Simulated Daily Stand‑ups by updating progress and identifying blockers.
* Completed Sprint Review by evaluating completed work.
* Reflected on improvements during a Sprint Retrospective
  ___________________________________________________________________________________________________________________________________

# Purpose of This Lab
This project demonstrates practical experience with Agile project coordination, including planning, documentation, backlog management, and workflow organisation. It reflects real‑world practices used by technical and cross‑functional teams.
